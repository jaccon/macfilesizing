Installation

The easiest way to install on macOS:

brew tap jaccon/tap
brew install macfilesizing

Then use it directly:

macfilesizing --source /path/to/directory

Install using pip:

pip install git+https://github.com/jaccon/jcfilesizer.git

Then use:

macfilesizing --source /path/to/directory

Clone and install manually:

git clone https://github.com/jaccon/jcfilesizer.git
cd jcfilesizer
pip install -r requirements.txt
python3 macFileSizing.py --source /path/to/directory

Usage Examples

Basic Usage

macfilesizing --source ~/Documents

Lists all files and directories in ~/Documents sorted by size.

Recursive Tree Mode

macfilesizing --source ~/Documents --tree

Recursively scans all subdirectories and ranks everything by size.

Generate Report

macfilesizing --source ~/Documents --report ~/report.txt

Saves the analysis to a report file.

Combined Options

macfilesizing --source ~/Downloads --tree --report ~/downloads_analysis.txt

Performs recursive analysis and exports the results to a file.

Command Line Arguments

Argument Required Description
--source Yes Path of the directory to be analyzed
--tree No Recursively scan all subdirectories
--report No Path to save the report file
